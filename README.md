# Vanilla

**Vanilla** is a lightweight, read-only menu platform for modern restaurants. It delivers a fast, digital menu directly to customers' phones through a single restaurant-wide QR code and short link.

## Key Features

* 📱 **Seamless QR Experience:** Customers scan one link to instantly view the full, up-to-date menu on any device.
* ⚡ **Blazing Fast:** Menu updates are cached and served instantly with zero load delay.
* 🕒 **Smart Availability:** Easily schedule custom menus or time-bound specials based on hours or days of the week.
* ☕ **Flexible Menu Items:** Supports item variants (like *Small/Large* or *Slice/Whole*) and clear allergen disclosures (*Contains* / *May Contain*).
* 🔒 **Read-Only Security:** Strictly designed for menu browsing—no clutter from checkout, payments, or account sign-ups.
